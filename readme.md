# Accounts Service
Mẫu source code base cho dự án nodejs + express 

# Install Guide

1. Cài các thư viện cần thiết

    Từ thư mục root của project chạy
```
npm i
```

2. Đổi tên `env.sample` => `.env`, cấu hình các thông tin phù hợp
```
rm env.sample .env
```

3. Chạy project

```
node index.js
```
